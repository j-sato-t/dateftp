# ftpクライアント

## 概要

- cliで操作するftpクライアント
  - 後からGUIも対応できるようにするために機能は全てpkg配下に実装する
  - CLIとして実行するmain関数はcmd/cli/main.goに実装する

## 環境構築

- miseを使用する
  - miseはインストール済みのものを使う
- goのバージョンは26.1.0以上
- 他のツールが必要になったらmiseでインストールする

## 要件

- 環境変数かフラグ指定で接続先を指定できる
  - 環境変数: FTP_HOST, FTP_PORT, FTP_USER, FTP_PASSWORD
  - フラグ: -host, -port, -user, -password
  - デフォルト値は無しで、環境変数とフラグどちらも指定がない場合はpanicさせる
  - フラグが優先される
- ダウンロードするルートとなるパスとダウンロード先ディレクトリを指定できる
  - 環境変数: FTP_ROOT_PATH, FTP_DOWNLOAD_DIR
  - フラグ: -root-path, -download-dir
  - FTP_ROOT_PATHと-root-pathはデフォルト値無しで、指定がない場合はpanicさせる
  - FTP_DOWNLOAD_DIRと-download-dirはデフォルト値無しで、指定がない場合はカレントディレクトリにする
  - フラグが優先される
  - FTP_ROOT_PATHの中にディレクトリがあった場合は、そのディレクトリの中身を再帰的にダウンロードする
- フラグは全て短縮形を用意する

## ダウンロード手順

1. FTP_ROOT_PATHのディレクトリ以降のディレクトリ構造をFTP_DOWNLOAD_DIR配下に再現する
2. ダウンロードするファイルのタイムスタンプを取得する
3. 1のディレクトリに `YYYY/MM/DD` 形式のディレクトリを作成する
4. 3のディレクトリにファイルをダウンロードする
5. ダウンロードしたファイルのタイムスタンプを2で取得したタイムスタンプに更新する

### ダウンロード実行例

- ダウンロードするファイルのパス : `/data/ftp/photo/20250508_12146.JPG`
- ダウンロード先ディレクトリ : `/home/jun/Downloads/dateftp/`
- FTP_ROOT_PATH : `/data/ftp/photo`
- FTP_DOWNLOAD_DIR : `/home/jun/Downloads/dateftp`
- ダウンロード完了後のファイルパス : `/home/jun/Downloads/dateftp/photo/2025/05/08/20250508_12146.JPG`

## すでにファイルがある場合

- ダウンロード先ディレクトリにすでにファイルがある場合は、タイムスタンプを比較して、ローカル側ファイルのタイムスタンプがftpサーバーのタイムスタンプより古い場合のみダウンロードする
